
# **Darth Buddha's Dank Theme**

## **Language Support**

- **Env**  - **.env**
- **Git** - **.gitignore**
- **Json** - **.json**
- **Python** - **.py**
- **Trunk Tree** - **.tree**

***

### **.vscode/settings.json**

Add some rulers

```

    "editor.rulers": [
        {
            "column": 0,
            "color": "#F0F8FF20"
        },
        {
            "column": 72,
            "color": "#F0F8FF20"
        },
        {
            "column": 79,
            "color": "#C8081550"
        }
    ],

```
